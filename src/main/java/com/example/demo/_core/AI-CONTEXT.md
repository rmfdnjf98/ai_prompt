# Core Utility

## 목적

프로젝트 전반에서 공통으로 사용되는 유틸리티 클래스 및 응답 규격 정의.

## 주요 파일

| 파일명 | 설명 |
| :--- | :--- |
| Resp.java | API 통일 응답 규격 (성공/실패 데이터 래핑) |

## 하위 디렉토리

- `utils/` - 도메인 독립적인 순수 유틸리티 모음

## AI 작업 지침

- 모든 API 응답은 `Resp.ok()` 또는 `Resp.fail()`을 통해 반환하여 일관된 JSON 포맷을 유지한다.
